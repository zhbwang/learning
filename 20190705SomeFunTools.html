1. Mac

1.1. Chrome

1.1.1. 快捷键

  • Shift + Command + <- 或者 ->：快速地切换上一个或下一个标签
  • Contral + Tab：切换到下一标签页
  • Command 1~9：直接选择对应地标签页
  • 空格键：向下滚动屏幕
  • Shift + 空格键：向上滚动屏幕
  • Command + N：新建标签页
  • Command + W：关闭标签页

1.1.2. Chrome 插件

参考：我装了 88 款 Chrome 扩展的浏览器里都有啥？ - 少数派

中英文对照翻译网页全文工具——彩云小译

推荐一款网页全文中英对照翻译软件——彩云小译（Lingocloud Interpreter）。

特色：支持Chrome等浏览器插件；每一段会有一段完整翻译，放在一段之后。最主要是翻译的结果非常通顺，不那么生硬。

官网：彩云小译 - 在线翻译 参考：兩款可以「中英對照翻譯網頁全文」的 Chrome 翻譯套件比較

Markdown Here

markdown here是一款非常高效易用的markdown格式工具。可以在诸多软件中支持markdown语法：

浏览器：Firefox（火狐），Chrome（谷歌），Safari（苹果）； 邮件客户端：thunderbird，Postbox，Icedove；

使用场景：

  1. 微信公众号排版
  2. 网页版邮件排版

官网：Markdown Here 参考：markdown|高效撰写微信公众号文章 - 简书

2. 写作

2.1. MineTime —— 建立21世纪的日历【弃用 | 不支持中文自然语言识别】

简介：在苏黎世联邦理工学院，我们研究如何将最新技术引入日常生活。 Minetime 是这个研究项目的一部分，该项目旨在构建一个现代的、多平台的、人工智能驱动的日历应用程序。

特色：

  • 与 Google 日历、 outlook 网站、 Microsoft Exchange、 iCloud 以及任何 CalDAV 服务一起工作: 这意味着你可以在 MineTime 中直接组织你所有的日历。
  • Minetime Insight 可以即时显示你在过去几个月中与同事见面的频率、事件被重新安排的次数以及更多信息。
  • Minetime是一个基于 Electron 的日历，它可以在 Windows，Mac 和 Linux 上作为桌面应用程序运行！

官网：MineTime | Get your new calendar for Win, Mac and Linux

2.2. WriteMapper——像做脑图一样来进行写作【弃用 | 可以用iThoughts替代】

特色：WriteMapper —— 脑图 + MarkDown 的结合体。

我们可以按照构思将文章按照结构进行拆分。它所生成的层层递进的子项对应的是我们在 MarkDown 语言中用到的 H1~H6。

完成大纲的梳理后我们可以点击任意一项进入当前段落的写作模式，这个时候界面会切换成大家常见的 MarkDown 写作模式，文本标签和操作和其他写作工具大同小异。我们可以随时切换进行分段写作。

相关工具：OmniOutliner。但是 OmniOutliner 的书写环境并不是太好，还是需要将大纲导出到 Ulysses 之类的写作工具中来继续写作。相比于 OmniOutliner，我们可以在WriteMapper随时对大纲的结构进行调整并进行相应的写作而无需来回跳转。

选择：如果你并不是一个对写作环境有较高要求的用户，希望能简单、快速的完成一份文档，WriteMapper 是一个可以尝试的新玩意。但如果你需要经常写一些较为复杂的文章， 我还是更推荐 OmniOutliner 加 MarkDown 的组合。

价格：WriteMapper 是一款付费应用，售价 26.95 美元。

官网：WriteMapper — Professional outlining software tool for your writing, using mind maps. 参考：像做脑图一样来进行写作 - 知乎
