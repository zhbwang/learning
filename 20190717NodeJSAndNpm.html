NodeJS + NPM 使用入门

NodeJS + NPM 使用入门

1. CentOS 7 中安装NodeJS

1.1. 下载

[root@bd129137 raw]# pwd
/usr/local/raw
[root@bd129137 raw]# wget https://nodejs.org/dist/v10.16.0/node-v10.16.0-linux-x64.tar.xz
[root@bd129137 raw]# tar xvJf node-v10.16.0-linux-x64

1.2. 配置

[root@bd129137 local]# pwd
/usr/local
[root@bd129137 local]# ln -s raw/node-v10.16.0-linux-x64 node

修改~/.bash_profile，然后使用source命令让配置生效。

# node
export NODE_HOME=/usr/local/node
export PATH=$PATH:${NODE_HOME}/bin

1.3. 验证

[root@bd129137 ~]# node -v
v10.16.0
[root@bd129137 ~]# npm -v
6.9.0

1.4. 修改 npm 的默认安装来源

查看默认npm的源地址

$ npm config get registry

修改源地址

npm config set registry https://registry.npm.taobao.org

验证是否成功

npm config get registry
或
npm info express
或
open ~/.npmrc

清除已设置的npm淘宝镜像

npm config delete registry
npm config delete disturl
或者
npm config edit
找到淘宝那两行,删除
